Common html tags for any html document

Here are some common html tags that could be useful for you when drafting a html document.

Tag name Tag in html Description
DOCTYPE <!DOCTYPE html> Indicating the language of the file (html). Use it at the top of your document.
html <html> Write all your html between this tag.
head <head> Heading of the page. In the head you can specify the title of the tab/website and link to your css files, fonts, etc. The text in <head> is not visible on the webpage for visitors.
title <title> The name of your website as visible in the tab of the browser.
link <link href='' ref='stylesheet'> Watch out: no closing tag. Links your html file to your stylesheet in css.
body <body> Insert your content here.
h1 <h1> To specify a heading. There is <h1> till <h6>.
p <p> For paragraphs.
div <div> To divide the html in separate blocks. You can also use <span> if you want to divide a sentence in blocks (e.g. a few words).
br <br> Page break (in essence: enter).
em and strong <em> & <strong> To use italics or bold in your text. Alternative: use css for this.
ul and ol <ul>& <ol> To make an ordered (OL) or unordered (UL) list.
Comment <!--comment--> To add a comment in your html text that is not visible in the website text.
Image <img src=''/> Look out: it's a self closing tag. You can add attributes to this as well, like alt (alternative text), width and height.
Video <video src=''> To add a video. You can add attributes to this as well, like controls (to get play/pause/etc.), width and height.
Audio <audio> <source src='' type='audio/mp3'> </audio> A bit of a weird one, since you have the opening- and closing tag, and a separate one nested in it for the source. You can also add attributes such as autoplay, loop, mute and controls.
Embedding content <embed src=''/> Look out: it's a self closing tag. Use it to embed gifs, videos etc. Source (like URL) required.
Hyperlinks <a href='link'> [text hyperlink] </a'> For internal links to files in the same folder, use ./[filepath]. For external links, use the URL. You can also link to #id's.

Other HTML tags for any html document

Other tags that should be mentioned, are semantic tags like header, nav, main, footer, section or article, section and aside, and for images, figure and figcaption. The html tags for creating a table are also very useful. You can check out those tags by pressing F12 on your browswer - check out the code that is used for creating the table above!
